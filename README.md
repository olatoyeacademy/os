# Olatoye Academy OS

A central education management platform for the Olatoye family — curating learning, activities and development for Elsie and Emma.

## What this is

Olatoye Academy is our umbrella for everything related to our children's education and development. This app brings together:

- **Homeschool** sessions we run directly
- **Kings Interhigh** — Elsie's online school
- **Kumon** — maths and reading programmes
- **Activities** — cheer, swimming and more

## Features (current)

- [x] Weekly schedule manager — combined view, colour-coded by child
- [x] Filter by child (Elsie / Emma) or by source
- [x] Add / remove sessions
- [x] Mark sessions as done / track attendance
- [x] Week navigation
